# Inbox competitor intelligence

Questa cartella raccoglie i delta generati dai run di spiru-ops.

## Regole
- Un file per run
- I run automatici possono scrivere qui
- Non scrivere direttamente nel `competitor-map.md`
- Dopo review, i file possono essere:
  - marcati come reviewed
  - oppure spostati in `../archive/`

## Contenuto atteso per ogni file run
- nuovi competitor emersi
- nuove evidenze su competitor già noti
- claim dubbi / da verificare
- cambi di pricing / canali / posizionamento
