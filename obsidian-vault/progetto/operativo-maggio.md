# Operativo maggio 2026 — Algavitae

Questo file serve come livello esecutivo tra il gantt mensile e i reminder/check-in.

## Regole operative
- Ogni settimana si selezionano i task realmente eseguibili dal piano mensile.
- Ogni giorno si sceglie 1 task non negoziabile + 1 task secondario.
- Conta solo ciò che produce un output concreto.
- Stati ammessi: `da fare`, `in corso`, `fatto`, `bloccato`, `rimandato`.
